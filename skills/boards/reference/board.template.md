# Project board

The single source of truth for planned and shipped work, rendered read-only at
`/admin/boards` (local dev only). Format: `## <Section>` headings; items as
`- [ ] **T-01 · Title** — body`. Status glyphs: `[ ]` to-do, `[~]` doing,
`[x]` done. Ids are stable — never renumber.
